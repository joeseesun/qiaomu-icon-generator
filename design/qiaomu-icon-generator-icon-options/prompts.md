# qiaomu-icon-generator Icon Prompts

Generated on 2026-06-21 with Codex built-in image generation. The goal was to create a public example for this skill itself, not a client project icon.

## Shared Constraints

Every candidate prompt used these constraints:

- One original square app icon candidate for `qiaomu-icon-generator`.
- Opaque square background.
- Bold centered symbol with generous safe padding.
- Readable at 32px for website favicon use.
- Usable as an iOS-style app icon master.
- No text, letters, numbers, pseudo-text, watermark, trademarks, or copied app icon compositions.
- Palette: charcoal black, warm amber, porcelain white, restrained cyan.

## Selected Direction

`option-05` was selected because it communicates the whole workflow at a glance: generate several choices, compare them, and produce one polished final icon. It also holds up better at 32px than the line-heavy export and magnifier concepts.
