# Qiaomu Music Icon Example

This folder contains the public example used by `qiaomu-icon-generator`.

The example is based on a real Qiaomu product, 乔木音乐网, and demonstrates both supported generation paths:

- `codex-bitmap/`: richer raster app-icon candidates generated with Codex built-in image generation and local icon reference guidance.
- `svg-cli/`: deterministic SVG candidates generated by QM Icon Studio CLI.
- `final-web/`: exported website icon sizes from the selected Qiaomu Music direction.
- `overview/`: README and social-preview images built from the real case.

## Selected Result

- Selected candidate: `codex-bitmap/selected-option-05-ios-1024.png`
- Direction: Abstract Q Disc
- Reason: best continuity with Qiaomu Music's black/gold favicon direction, while staying readable at 32px.

## SVG CLI Command

```bash
node "$HOME/Documents/qm-icon-studio/cli/qm-icon-options.mjs" \
  --name "Qiaomu Music" \
  --query music \
  --count 8 \
  --out design/qiaomu-music-svg-cli-options \
  --offline
```

The CLI outputs editable SVG files and `contact-sheet.svg`. PNG previews in this folder are documentation previews rendered from those SVGs.
